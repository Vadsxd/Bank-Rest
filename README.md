<h1>Система Управления Банковскими Картами</h1>

<h2>Описание</h2>
  <p>
    API для управления картами и администрирования пользователей/карт
  </p>

<h2>Технологии</h2>
  <p>
    Java 21, Spring Boot 4.0.2, Spring Security, Spring Data JPA, PostgreSQL 16.0, Liquibase, Docker 4.49, JWT, Swagger 3.0
  </p>

<h2>Запуск</h2>

1) git clone https://github.com/Vadsxd/Bank-Rest.git

2) cd Bank-Rest
   
3) docker-compose up -d

<h2>Swagger:</h2>
* http://localhost:8080/swagger-ui/index.html#/
